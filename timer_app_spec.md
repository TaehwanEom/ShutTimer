# ShutTimer — 스펙 문서

---

## 기본 정보
- 앱 이름: ShutTimer
- 플랫폼: iOS + Android
- 개발: Expo (React Native) + Claude Code (Antigravity)
- 서버: 없음
- 로그인: 없음
- 데이터 저장: 로컬 (AsyncStorage)

---

## v1 범위 (확정)

### 화면 구조
1. **홈 화면** — 다이얼 링 타이머 + 즐겨찾기 목록 + 시작 버튼
2. **타이머 실행 화면** — 다이얼 링 카운트다운 + 일시정지/취소
3. **알람/미션 화면** — 알람 + 종료 방식 실행

### 타이머
- 60분 고정 (v1)
- 다이얼 링 형태 (도넛 스타일, Time Timer 스타일)
- 흰 배경 + 빨간 링
- 백그라운드: 로컬 푸시 알림 예약 방식 (iOS/Android 통일)

### 즐겨찾기 알람
- 기본 제공: TV시청, 목욕, 독서, 공부, 양치, 놀이, 게임, 요리
- 사용자가 직접 편집 가능 (추가/삭제/순서변경)
- 로컬 저장

### 알람 종료 방식 (설정에서 선택)
- 그냥 끄기 (버튼 탭)
- 흔들어 끄기 (폰 흔들기)
- 사진 찍어 끄기

### 알람 사운드
- 커스텀 mp3 사용
- Suno로 직접 제작 (상업용 가능)

### 광고
- AdMob 적용
- 홈 화면 하단 배너
- 미션 완료 후 인터스티셜 전면 광고 (단가 높은 광고 자동 배치)

### 인앱 결제
- 광고 제거 일회성 구매
- Apple/Google 계정 기반, 앱 재설치 후 복원 가능
- 복원 버튼 앱 내 포함

- 무음/진동 선택

## 의견 (미확정 — 검토 필요)

- **반복 타이머**: 매일 같은 시간 자동 시작 — 루틴 관리용으로 유용할 수 있음
- **홈화면 위젯**: 앱 열지 않고 바로 타이머 시작 — 사용 빈도 높아질 수 있음
- **타이머 기록**: 언제 뭘 했는지 로컬 저장 — 습관 추적 기능으로 확장 가능
- **미션 완료 SNS 공유**: 완료 이미지 생성 후 공유 유도 — 바이럴 마케팅 효과 가능
- **커스텀 테마**: 다크모드, 컬러 변경 — 무료 제공으로 차별화 요소로 활용 권장

## 기술 스택

| 항목 | 라이브러리 |
|------|-----------|

## 개발 순서 (v1)
1. Expo 프로젝트 초기 세팅
2. 다이얼 링 타이머 UI
3. 카운트다운 로직 + 백그라운드 알람
4. 알람 종료 방식 (탭/흔들기/사진)
5. 즐겨찾기 편집 기능
6. AdMob 광고 위치 세팅
7. 인앱 결제 (광고 제거)
8. 실기기 테스트 (Expo Go)
9. 앱스토어 배포 준비
